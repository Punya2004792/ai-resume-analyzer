# 🚀 AI Resume Analyzer

## 📌 Overview

AI Resume Analyzer is a smart web application that analyzes resumes and compares them with job descriptions. It provides a match score, identifies missing skills, and suggests improvements to increase job selection chances.

---

## 🎯 Problem Statement

Many students and job seekers are unaware of how well their resume matches a job description, leading to low chances of getting shortlisted.

---

## 💡 Solution

This project uses AI techniques to:

* Extract text from resumes
* Compare it with job descriptions
* Generate a match score
* Identify skill gaps
* Provide improvement suggestions

---

## ✨ Features

* 📄 Upload Resume (PDF)
* 📊 Match Score Calculation
* ✅ Skills Identification
* ❌ Missing Skills Detection
* 📌 Smart Suggestions

---

## 🛠 Tech Stack

* Python
* Streamlit
* Scikit-learn
* PyPDF2

---

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app.py

---

## 📊 Output

* Resume match percentage
* List of matched skills
* Missing skills
* Suggestions to improve resume

---

## 🎯 Use Case

Helps students and job seekers improve their resumes and increase their chances of getting selected in interviews.
